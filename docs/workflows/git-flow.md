# Git Flow

Branch naming:
- `main`: protected production branch
- `<name>` for personal branches (example: `sadik`, `sabri`, `jaffal`)

Standard flow:
1. Pull latest `main`.
2. Create/switch to your branch.
3. Commit small logical changes.
4. Push branch to GitHub.
5. Open PR to `main`.
6. Require review before merge.
